Some information in this page that you should know!


This page utilizes javascript's window.localStorage object, which basically saves values in your browser. This means any updates you will make using Edit Mode will remain, even when you reload the page, until you click the Reset button.


    This Edit Mode option, which you can see on the navbar, allows you to change few things in this page such as background color, text color, and typography.



    This is the Reset button. Clicking this will clear the values stored in the window.localStorage object which will reset the page to its default settings.
